ระบบคลังสูตรอาหาร

1. ส่วนเข้าใช้งาน
   
    1.1 หน้า Landing Page / Home (สำหรับผู้เยี่ยมชม): นำเสนอจุดเด่นของแอป, สูตรอาหารยอดฮิตประจำสัปดาห์ และปุ่มชักชวนให้สมัครสมาชิก
   
    1.2 หน้า Login / Register: สำหรับเข้าสู่ระบบ (รองรับ Email และ Social Login เช่น Google/Facebook)

2. ส่วนแดชบอร์ดหลัก (Main Dashboard):
   
    2.1 หน้า Dashboard (หน้าแรกเมื่อล็อกอิน):
        สรุปภาพรวม (สูตรที่บันทึกไว้, สูตรที่เพิ่งเข้าดู)
        "เมนูแนะนำจากวัตถุดิบที่มี" (เชื่อมโยงกับระบบคลังวัตถุดิบ)
        การแจ้งเตือน (เช่น วัตถุดิบในคลังกำลังจะหมดอายุ)

3. ส่วนจัดการสูตรอาหาร (Recipe Management):
   
    3.1 หน้าค้นหาและแคตตาล็อก (Search & Browse):
        ระบบค้นหาด้วย ชื่อ, ประเภท (ต้ม, ผัด, แกง, ทอด), สัญชาติอาหาร หรือ ค้นหาด้วยวัตถุดิบ (Filter by Ingredients)

    3.2 หน้ารายละเอียดสูตรอาหาร (Recipe Detail):
        รูปภาพ / วิดีโอประกอบ
        รายการวัตถุดิบและสัดส่วน (สามารถคำนวณปรับสัดส่วนตามจำนวนเสิร์ฟได้อัตโนมัติ)
        ขั้นตอนการทำ (ทีละสเต็ป)
        ปุ่ม "ทำอาหารเมนูนี้" (ลิสต์รายการวัตถุดิบที่ต้องใช้)
        ปุ่มเพิ่มลงรายการซื้อของ (Add to Shopping list)

4. ส่วนคอมมูนิตี้และผู้ใช้ (User & Community)
   
    5.1 หน้าโปรไฟล์:
        "สูตรที่ชื่นชอบ" (Favorites / Bookmarks)

    5.2 หน้าตั้งค่า:
        จัดการข้อมูลส่วนตัว, รหัสผ่าน, และการแจ้งเตือน (เช่น เตือนให้ไปจ่ายตลาด)

5. ฟีเจอร์เสริม
    
    6.1 Meal Planner (หน้าตารางวางแผนมื้ออาหาร): ปฏิทินรายสัปดาห์สำหรับลากสูตรอาหารไปวาง เพื่อวางแผนว่าแต่ละวันจะกินอะไร
   
    6.2 Nutrition Info (ข้อมูลโภชนาการ): คำนวณแคลอรี่รวมของแต่ละสูตรอาหาร
